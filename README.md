# Dataset Description

This dataset of the entire French Transmission-Grid topology is called RTE7000 or RTE7K (stands for 7,000 nodes representing voltage levels from 400 kV to 63 kV) and is now hosted at [https://huggingface.co/datasets/OpenSynth/RTE700](https://huggingface.co/datasets/OpenSynth/D-GITT-RTE7000-2021)

If you have any questions about this dataset, please use our [discussion space on HuggingFace](https://huggingface.co/datasets/rte-france/RTE7000/discussions?status=open&type=discussion)
